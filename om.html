Caroline Schytte - Kompetencer

1. semester

På 1. semester har jeg lært en masse om kodning, design og brugeroplevelser.

Ved selve kodningen har jeg lært om det mest basale indenfor HTML, CSS og JS.

I Visual Studio Code har vi lært at sætte HTML ind med div og sections, hvordan class og Id fungerer. I CSS har vi lavet custom settings og lært en masse indenfor grid, flexbox, keyframes animationer og hover funktionen.

I Figma har jeg lært at lave komponenter og interaktive prototyper.
